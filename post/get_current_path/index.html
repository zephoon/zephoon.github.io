get_current_path

Difference between the '.' and the __file__.

  1. get the current execute path no matter where the code in any python file is
from pathlib import Path
CURRENT_FILE_PATH = Path('.')
  1. get the current python file path no matter where the execute path is
from pathlib import Path
CURRENT_FILE_PATH = Path(__file__)

So the __file__ is more safe to use in the python project.😃😃😃

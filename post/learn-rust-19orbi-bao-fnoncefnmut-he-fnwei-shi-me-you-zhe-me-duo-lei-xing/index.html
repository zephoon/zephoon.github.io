Learn Rust 19｜闭包：FnOnce、FnMut和Fn，为什么有这么多类型？

2022-08-15 1 min read # Rust

《陈天· Rust 编程第一课》

  1. 闭包是将函数，或者代码和其环境一起存储的一种数据结构。
  2. 闭包的大小跟参数和局部变量无关，只跟捕获的变量有关。
  3. 闭包的效率非常高。
  4. 闭包捕获的变量都存储在栈上，没有堆内存分配。每个闭包都是一个新的类型，Rust 不需要额外的函数指针来运行闭包。闭包的调用效率和函数调用几乎一致。

anaconda使用笔记

· 2020-04-01 · # anaconda # conda

安装

  1. 下载
    https://www.anaconda.com/distribution/#download-section

  2. 拷贝到对应的主机，然后安装路径可以选择/home/anaconda3

  3. 如果需要避免污染.bashrc, 安装过程可以取消对.bashrc的修改。

  4. 执行conda，直接执行/home/anaconda3/bin/conda

查看conda基本信息

conda info

查看虚拟环境

conda env list

激活环境

conda activate env_1
conda deactivate env_1
check the anaconda version
conda list anaconda
update
conda update conda
conda update anaconda
conda update all  # would update anaconda-navigator, anaconda-project, anaconda-client

离线安装package

pip install local_package.tar.gz
conda install --user-local local_package.tar.gz

查询python版本

conda search python
conda create -n py37 python=3.7
conda env remove -n py37

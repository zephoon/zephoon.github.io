windows搭建hadoop, spark, pyspark开发环境

· 2020-06-13 · # spark # python # 大数据 # 环境配置

新公司配了一台Thinkpad T480s，工作不能用自己使用多年的MacbookPro， 很多使用习惯从头再来，相当折腾。😭😭😭

  1. 基本环境配置信息
Item Version
window 10
python 3.7.6
conda 3.8.1
pyspark 2.4.4
spark 2.4.4
hadoop 2.7.7
JDK 1.8
  1. 下载

安装方式：直接解压到对应的路径下面，例如C盘的根目录下。

Item Remark
apache spark spark-2.4.4-bin-hadoop2.7.tgz 包含pyspark包，可直接进行安装。
hadoop
winutils 解压之后找到对应的版本，用里面bin替换hadoop的bin，这样才能在windows下运行hadoop。
  1. 配置环境变量
Item Path
SPARK_HOME: C:/spark/spark-2.4.4-bin-hadoop2.7
HADOOP_HOME: C:/hadoop/hadoop-2.7.0
JAVA_HOME: C:/PROGRA~1/Java/jdk1.8.0_62
Path: Path; %SPARK_HOME%/bin;%HADOOP_HOME%/bin;%JAVA_HOME%/bin;

  1. 处理配置文件
    %HADOOP_HOME%/etc

    1. core-site.xml
    <configuration>
  <property>
     <name>fs.default.name</name>
     <value>hdfs://localhost:9000</value>
  </property>
</configuration>
    1. hdfs-site.xml
    <configuration>
  <property>
     <name>dfs.replication</name>
     <value>1</value>
  </property>
  <property>
     <name>dfs.namenode.name.dir</name>
     <value>file:/C:/hadoop/hadoop-2.7.7/data/namenode</value>
  </property>
  <property>
     <name>dfs.datanode.data.dir</name>
     <value>file:/C:/hadoop/hadoop-2.7.7/data/datanode</value>
  </property>
</configuration>
    1. mapred-site.xml
    <configuration>
    <property>
    <name>mapreduce.framework.name</name>
    <value>yarn</value>
    </property>
</configuration>
    1. yarn-site.xml
    <configuration>
    <property>
    <name>yarn.nodemanager.aux-services</name>
    <value>mapreduce_shuffle</value>
    </property>
    <property>
    <name>yarn.nodemanager.aux-services.mapreduce.shuffle.class</name>
    <value>org.apache.hadoop.mapred.ShuffleHandler</value>
    </property>
    <property>
        <name>yarn.resourcemanager.webapp.address</name>
        <value>127.0.0.1:8088</value>
    </property>
    <property>
        <name>yarn.resourcemanager.hostname</name>
        <value>localhost</value>
    </property>
</configuration>

  2. 启动测试

    $ cd %HADOOP_HOME%/sbin
$ hdfs namenode -fromat
$ start-dfs.cmd
$ start-yarn.cmd
$ hdfs dfs -ls .
$ hdfs dfs -mkdir /user/hadoop
$ hdfs dfs -ls /user
$ hdfs dfs -put /local/text /user/hadoop
$ hdfs dfs -cat /user/hadoop/text

  3. 查看hadoop自带的web控制台

    1. 资源管理: http://localhost:8088/
    2. 节点管理: http://localhost:50070/

  4. 安装pyspark
    可以自行设置清华anaconda镜像源

    conda install pyspark

  5. 代码测试

    import findspark 
findspark.init() 
import pyspark # only run after findspark.init() 
from pyspark.sql import SparkSession 
spark = SparkSession.builder.getOrCreate()  
df = spark.sql('''select 'spark' as hello ''') 
df.show()

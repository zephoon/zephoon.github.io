window配置pyspark

· 2020-01-19 · # spark # python # 大数据 # 环境配置

新公司配了一台Thinkpad，工作不能用自己使用多年的MacbookPro， 很多使用习惯从头再来，相当折腾。😭😭😭

  1. 基本环境信息
window 10
python 3.7.6
conda 3.8.1
  1. 下载

安装方式：直接解压到对应的路径下面，例如D盘的根目录下。

apache spark

winutils

  1. 配置环境变量
SPARK_HOME: C:/spark/spark-2.4.4-bin-hadoop2.7
HADOOP_HOME: C:/hadoop/winutils/hadoop-2.7.0
  1. 安装pyspark
    可以自行设置清华anaconda镜像源
conda install pyspark
  1. 代码测试
import sys
import os
from pathlib import WindowsPath

SPARK_HOME_PATH = "C:/spark/spark-2.4.4-bin-hadoop2.7"
HADOOP_HOME_PATH = "C:/hadoop/winutils/hadoop-2.7.0"
os.environ['SPARK_HOME'] = SPARK_HOME_PATH
os.environ['HADOOP_HOME'] = HADOOP_HOME_PATH
if not SPARK_HOME_PATH in sys.path:
    sys.path.append(SPARK_HOME_PATH)
if not HADOOP_HOME_PATH in sys.path:
    sys.path.append(HADOOP_HOME_PATH)

import findspark 
findspark.init() 
import pyspark # only run after findspark.init() 
from pyspark.sql import SparkSession 
spark = SparkSession.builder.getOrCreate()  
df = spark.sql('''select 'spark' as hello ''') 
df.show()

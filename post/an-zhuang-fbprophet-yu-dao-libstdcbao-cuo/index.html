安装fbprophet遇到libstdc++报错

· 2019-06-28 · # 环境配置 # 机器学习
环境：
  • centos7.2
  • python3.6.7
过程

普通pip install fbprophet出现gcc报错 利用conda install -c anaconda fbprophet安装过程没问题，但是在使用就出现报错

from fbprophet import Prophet
>>> ImportError: /lib64/libstdc++.so.6: version 'GLIBCXX_3.4.21' not found
解决处理

排查跟gcc版本有关，编译安装太花时间，scl安装还是报同样的错误。 conda可以安装libgcc， conda install -c anaconda libgcc 安装之后 ls ~/anaconda3/lib | grep libstdc++ 查询发现有对应的libstdc++.so.6.0.21 然后需要设置环境变量 export LD_LIBRARY_PATH=$LD_LIBRARY_PATH:~/anaconda3/lib/ 执行 from fbprophet import Prophet 完美没报错！！！

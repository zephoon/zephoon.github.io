Git config for personal and work

· 2020-12-03 · # git

Backgroud:

When i work in out company network, there is another account to login version control system such as bitbucket. But my personal common version control system is git from github. After all, these two technologies could use the git as the tool. So we could use these two accounts infos in different config file from git.

Solutions:

  • The content for the main entry, my personal git config and include the work config.
[user]
    name = XXXX
    email = XXXX@XXXX.com

[http "https://github.com"]
	proxy = http://XXXX:XXXX

[https "https://github.com"]
	proxy = http://XXXX:XXXX

[include]
    path = C:/Users/XXXX/Projects/.gitconfig;
  • The content for my work git config file.
[user]
    name = XXXX
    email = XXXX@XXX.com
[http]
    sslCAInfo=C:/Users/XXX/Projects/XXX.pem

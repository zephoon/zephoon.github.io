V2rayX in macOS

· 2019-11-24 · # v2ray # macos

ss和ssr经常被封😷😷😷，最终只能多花点时间实施v2ray方案了😎😎😎。

环境

  • macOS: 10.15.1
  • V2rayX: 1.5.1

配置信息

  • V2ray订阅链接（自行搭建或者直接购买👻👻👻）
  • 服务器分享vmess链接（自行搭建或者直接购买👻👻👻）

配置流程

  1. 配置订阅链接
    • Configure... -> Advanced -> Subsription
  2. 配置服务器分享vmess链接
    • Configure... -> Import -> Import from other links
  3. Server -> Update subsription
  4. 一般选择【PAC Mode】
  5. v2ray-core load 开启服务
  6. 查看view log
    2019/11/24 22:25:09 tcp:127.0.0.1:54781 accepted tcp:0.client-channel.google.com:443 
2019/11/24 22:25:09 tcp:127.0.0.1:54780 accepted tcp:safebrowsing.googleapis.com:443
  7. OK! 😀😀😀

😺😺😺good luck！

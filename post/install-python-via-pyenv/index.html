pip 安装本地包
Having requirements in requirements.txt and egg_dir as a directory
you can build your local cache:
$ pip download -r requirements.txt -d eggs_dir
then, using that "cache" is simple like:
$ pip install -r requirements.txt --find-links=eggs_dir
$ pip install -r plist.txt --find-links=/home/log-analyse/packages/
$ pip uninstall -r plist.txt -y
$ rm -rf ~/.cache/pip/
$ tar xvf XXX.tar ./
$ tar -cvf packages.tar ./packages ./plist.txt
